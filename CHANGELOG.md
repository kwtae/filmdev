# 업데이트 기록 (Changelog)

## [v1.1.0] - 2026-03-23
### ✨ 기능 추가 (Features)
- **Massive Preset DB (프리셋 데이터베이스 고도화):** 텅 빈 앱에서 최초 시작 시 `Load Preset` 버튼을 누르면 단일 샘플이 아닌, 글로벌 스탠다드 현상 시간표(Massive Dev Chart)의 가장 대중적인 정밀 데이터 6종(Tri-X D-76, HP5 HC-110, Portra C-41, Fomapan Rodinal, Cinestill 800T 푸쉬 현상 등)이 초 단위 교반 세팅값과 함께 일괄 적재(`bulkPut`)되는 기능 완비.
- **자동화 현상 일지 (Logbook & History):** 현상 작업 완료 후 `Complete & Save Log` 클릭 시, 해당 날짜의 레시피 제원 및 센서 온도, 코멘트 등을 IndexedDB `logs` 하위 스키마에 자동 적재. 네비게이션 탭의 전용 기록장(Logbook) 페이지에서 시간 순으로 열람/삭제 가능.
- **햅틱 모터 제어 (Haptic Feedback):** Web API `navigator.vibrate()`를 호출하여 단순 오디오 비프음만이 아닌 촉각적 피드백 제공 (화면 탭 및 현상 알람 발생 시 디바이스 물리 진동 발생).
- **레시피 공유 기능 (Share/Export):** 홈 화면 레시피 리스트에 직접 공유(Share2) 버튼을 부착하여, `navigator.share` (모바일 OS 기본 공유 모달) 또는 클립보드를 통해 텍스트화된 레시피 정보를 동료들에게 즉시 송신 가능.
- **위키 모듈 탭(Tab) 4분할 병합 아키텍처:** 위키 페이지를 (실전 팁) / (화학적 원리) / (온도 계산기) / (실패 부검소) 4가지 탭으로 디자인을 180도 전면 개편하고 30개가 넘는 전문가용 컴파운드 지식 및 웹 기반 이슈 트러블슈팅 자료 적재.
- **오류 분석 데이터실 (Troubleshooting Clinic):** 망상 현상, 퍼플 캐스트, 스팟 얼룩, 블랭크 샷 등 세계의 암실 유저들이 가장 많이 질문하는 실패 사례들을 증상과 화학적 분자 원인 기반으로 설명하는 에러-부검소를 별도 구축함.
- **오버레이 일시정지 (PAUSE Overlay Tracking):** 사용자의 액체를 묻은 손이나 장갑을 착용한 손으로도 명료히 통제할 수 있도록, 일시 정지 시 화면 상단을 뒤덮는 거대한 빨간색 재생(`Play`) 아이콘 블라인트를 출력하며 텍스트 애니메이션(핑/바운스)을 적용하여 안전사고(망각) 방지. 타이머 구동 중 화면 전체에 아주 투명한 `Pause` 아이콘 힌트를 추가함.
- **레시피 삭제(Deletion) UI/UX:** 레시피 에디터 모드에서 기존 제작된 레시피를 아이디 기반으로 데이터베이스에서 영구 삭제할 수 있는 `Trash2` 버튼 추가 보완.
- **종료 정리 체크리스트 (POST-CHECKLIST):** 기존의 단순 버튼으로 종료되던 `FINISHED` 단계를 고도화하여 현상액 수세, 건조, 약품 폐기 및 장비 세척 절차(4단계)를 유저가 명시적으로 확인한 후에만 종료-로그저장이 이루어지도록 안전 루틴 강화.
- **위키(Wiki) 고도화 (동적 온도 계산기 및 팁 20개):** 아레니우스 방식을 차용한 `t2 = t1 * exp(0.081 * (T1 - T2))` 기반의 동적 온도-시간 보상(Compensation) 계산기 반응형 CSS 레이아웃 버그 수정 및 레코드를 20개 이상의 최고급 정보(HC-110, XTOL 특성, 크로스 프로세싱 등)로 대거 확충함.
- **체크리스트(PRE-CHECK) UX 초고도화:** 기존 3단계에서 4단계 체크리스트로 확장, 체크 항목마다 명시적인 번호를 부과하고 오디오 컨텍스트 락(Unlock)을 직접 테스트해볼 수 있는 'Test Ping' 버튼을 인라인으로 추가하여 현상 직전 스마트폰의 소리 작동 여부를 강제 검증하도록 설계 변경.
- 하단 네비게이션 액션바 버튼의 불필요한 크기변화(scale), 트랜지션 및 핑 애니메이션을 제거해 조작의 명료성(명시성)을 올림.
- **BLE(Bluetooth Low Energy) 센서 온도 모니터링:** 아날로그/디지털 무선 온도 센서 개입을 위한 `Web Bluetooth API` 통신 스크립트 구축 및 타이머 UI(`ActiveProcessScreen.tsx`)에 실시간 온도 모니터링 연동.
- **국제화(i18n):** 글로벌 Zustand 상태(`langStore`)를 도입하여 한국어(ko)와 영어(en)간 즉각적인 UI 텍스트 전환(Toggle) 기능 추가.
- 진행 중인 타이머 사이클을 즉시 중단하고 다음 단계 과정으로 넘어갈 수 있는 **`Skip Step ⏭` (단계 건너뛰기) 기능** 추가.
